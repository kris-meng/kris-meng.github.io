Things I've built

001 winning project

TMS Stimulation Visualizer

Patient-specific simulation framework modelling the effects of Transcranial Magnetic Stimulation on functional brain connectivity. 47.8% of simulated patients showed improved connectivity, aligning with clinical literature (40% response rate).

Connectomics 3D Visualization LLM Analysis Evolved Tech Hackathon
002 finalist

Mental Health AI Chatbot Guardrails

Stress-tested a mental health chatbot for safety vulnerabilities. Generated synthetic dialogue datasets via LLM multi-agent self-play. Fine-tuned mBERT classifiers for input guardrails with a secondary LLM judge for uncertain predictions.

LLMs mBERT Synthetic Data Mila Hackathon
003

ML Reproducibility Challenge: SimMTM

Reproduced SimMTM — a channel-independent transformer encoder with masked reconstruction pretraining and downstream forecasting fine-tuning. Implemented geometric masking, similarity matrix computation, and manifold + reconstruction loss.

Transformers Time-series PyTorch Self-supervised

Vision Transformer with CNN Feature Extractor

Hybrid CNN–ViT architecture for ground-based cloud image classification. Achieved 86.2% test accuracy, outperforming both standalone CNN and ViT baselines.
